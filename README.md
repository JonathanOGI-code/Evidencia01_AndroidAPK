# Portafolio del Módulo

Este repositorio contiene los proyectos y documentos desarrollados a lo largo del módulo de **Desarrollo de Aplicaciones Móviles**.  
Cada proyecto representa una etapa clave del aprendizaje, aplicando los conceptos vistos en clase mediante el uso de **Android Studio con Java**.

Además, este repositorio agrupa los proyectos individuales como **submódulos**, permitiendo un mejor control de versiones y una mejor organización.

---

## Proyectos Incluidos

### 1. HolaMundoApp
**Descripción:**  
Primer proyecto del módulo, cuyo objetivo fue familiarizarse con el entorno de desarrollo Android Studio.  
La aplicación muestra un mensaje simple en pantalla: “Hola Mundo”.

**Experiencia:**  
Aunque sencillo, fue un punto de partida esencial para comprender la estructura de un proyecto Android, los archivos XML de interfaz y el uso de `MainActivity.java`.

---

### 2. LayoutApp
**Descripción:**  
Proyecto enfocado en el uso y diseño de diferentes tipos de **Layouts** (LinearLayout, RelativeLayout, ConstraintLayout, etc.) para estructurar interfaces visuales.  
Se exploró la jerarquía de vistas y la alineación de elementos.

**Experiencia:**  
Este proyecto permitió entender cómo los layouts afectan la experiencia del usuario y cómo combinarlos para lograr interfaces más atractivas y funcionales.

---

### 3. TouchApp
**Descripción:**  
Aplicación creada para trabajar con eventos táctiles y la detección de gestos dentro de la pantalla.  
Implementa interacciones básicas como toques, desplazamientos y pulsaciones largas.

**Experiencia:**  
Permitió comprender cómo el sistema Android maneja los eventos del usuario y cómo responder a ellos mediante controladores (`onTouchEvent`, `GestureDetector`, etc.).

---

### 4. FragmentsApp
**Descripción:**  
Proyecto enfocado en la creación y manejo de **Fragments**, componentes reutilizables de la interfaz en Android.  
Se implementó una navegación básica entre distintos fragmentos dentro de una misma actividad.

**Experiencia:**  
Este proyecto consolidó el uso de fragmentos y el manejo de transacciones dinámicas. Fue un paso importante hacia la construcción de interfaces modulares y escalables.

---

## Conclusión General

Este portafolio refleja la evolución del aprendizaje durante el módulo, desde los fundamentos hasta el uso de estructuras más avanzadas en Android.  
Cada proyecto ayudó a reforzar habilidades técnicas, la comprensión del ciclo de vida de una aplicación y la importancia de la interfaz de usuario.

También se aplicaron buenas prácticas de control de versiones con Git y GitHub, integrando cada proyecto como submódulo dentro de este repositorio principal.

---

## Tecnologías Utilizadas

- Android Studio (Java)  
- XML para diseño de interfaz  
- Git y GitHub (submódulos)
